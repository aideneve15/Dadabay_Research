# Project Title

## Overview
Scientific Question & Approach
This project investigates chemical and metabolic variation in sagebrush using liquid chromatography–mass spectrometry (LC–MS) data. The central goal is to identify patterns in metabolite profiles across samples and conditions, and to determine which features contribute most strongly to observed biological differences.

Raw LC–MS data are preprocessed to remove noise, correct for retention time variability, and normalize signal intensities prior to analysis. Feature-aligned datasets are then explored using multivariate statistical and machine learning approaches, including dimensionality reduction and clustering, to characterize structure in the data and assess relationships among samples.
